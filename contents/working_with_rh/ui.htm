Menu and Top Toolbar

The Menu and Toolbar are much simpler than before. The options appear above the panels when you need them.

On the left are icons to save, undo and redo, find a topic and find and replace.


Above the authoring area are the icons you will need as you work. They will change according to what you are working on.

 
On the right are icons to change your view, generate and hide the right panel.

  1. Edit replaces Design in earlier versions of Adobe RoboHelp.

  2. Source replaces HTML in earlier versions of Adobe RoboHelp.

  3. Preview lets you see how the topic will appear in a browser.
    The preview opens in the Authoring Area so no popup any more, it opens in the same panel instantly.

  4. Quick Generate opens a list of your outputs.
    Those shown with blue text have been generated previously .
    If you don't need to change the settings, you just tick the outputs you want to generate and click Generate.
    The cog icon lets you access all your outputs if you do need to edit the settings first or create a new output first.
    This replaces Batch Generate in earlier versions.

  5. Hide Right Panel lets you hide the panel to give you more space to work in.

In earlier versions of Adobe RoboHelp if you selected text in Edit View and changed to Source, the source code was also selected. In Adobe RoboHelp 2019 you can also select text in Source View and it will be selected when you return to Edit View.

Left Toolbar

The left toolbar controls what is displayed in the panel alongside.

  • Contents

  • Table of Contents

  • Index

  • Variables

  • Condition Tags

  • Snippets

  • Glossary

  • Reports

If you click the chevron icon, text labels display. Useful while you familiarize yourself with the icons. There is also bubble text when you hover over an icon. The selected icon is blue and the panel alongside displays the information related to your selection, Contents in the image above.
